# Racku Voyage ✦

**Luxury hotel booking platform** powered by [LiteAPI](https://liteapi.travel).

## About
Racku Voyage is a commission-based hotel booking app built on Base44. It earns 15% margin on every hotel booking made through the platform.

## Pages
- **Home** — Landing page with destination search
- **Search** — Hotel search results with live rates
- **Hotel** — Hotel detail page with room selection
- **Book** — Booking form and confirmation
- **Dashboard** — Commission and booking tracking dashboard

## Tech Stack
- React (Base44 mini-app framework)
- LiteAPI v3.0 for hotel data and rates
- Base44 backend for booking records
- 15% commission margin on all bookings
